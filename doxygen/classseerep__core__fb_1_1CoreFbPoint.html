Detailed Description

This class is the center piece between the gRPC interface, the core and the hdf5-io for points.

The functions of this class are called by the corresponding gRPC services. The data storing and loading is done directly via the hdf5-io-fb. When adding new data the needed information for the indices are disclosed to the core. When data is queried the core is consulted to get the UUIDs of the data answering the query. The data is then loaded from this class via the hdf5-io.
