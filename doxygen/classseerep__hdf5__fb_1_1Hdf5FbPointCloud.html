Returns the offset for a rgb(a) channel.

The rgb(a) color information of a point is packed into a single uint32. We therefore need to add an addtion offset to the start of the rgb(a) channel.

Parameters
channel_nameName of the color channel to retrieve the offset for.
base_offsetThe offset to the start of the rgb(a) channel.
is_big_endianThe endianness of the point cloud.
Returns
The offset of the specified channel.
Exceptions
runtime_errorwhen the channel_name is not "r", "g", "b" or "a".
