Detailed Description

This is the class for a single instance.

It handles all data regarding instances. The attributes of the instances are queryable. And the attributes can also be written. The changes are persisted in the HDF5 file via the corresponding IO class.

The UUID of all data showing the instances are stored and can be easily queried.
