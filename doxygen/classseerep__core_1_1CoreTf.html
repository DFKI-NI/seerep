Detailed Description

This is the class handling the TF buffer.

TFs are stored in a tf2::BufferCore. TFs can be added to this buffer and queries from one frame to another can be queried for a specific point in time. AABBs and spatial queries can also be transformed directly in the map frame of the project via public methods.
