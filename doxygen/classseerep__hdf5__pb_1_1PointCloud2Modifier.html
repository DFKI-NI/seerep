  ... 
)

Function setting some fields in a PointCloud and adjusting the internals of the PointCloud2.

Parameters
n_fieldsthe number of fields to add. The fields are given as triplets: name of the field as char*, number of elements in the field, the datatype of the field, the datatype of the elements in the field

E.g, you create your PointCloud2 message with XYZ/RGB as follows: 

*   setPointCloud2Fields(cloud_msg, 4, "x", 1, seerep::PointField::FLOAT32,
*                                              "y", 1, seerep::PointField::FLOAT32,
*                                              "z", 1, seerep::PointField::FLOAT32,
*                                              "rgb", 1, seerep::PointField::FLOAT32);
*

WARNING: THIS DOES NOT TAKE INTO ACCOUNT ANY PADDING For simple usual cases, the overloaded setPointCloud2FieldsByString is what you want.
